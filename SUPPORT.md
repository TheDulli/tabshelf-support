# Support

## Scope

This repository handles public support for TabShelf:

- Bug reports
- Feature requests
- Usage questions

## Response targets

- Initial triage target: within 7 days
- Fix timelines are not guaranteed

## Before opening an issue

1. Confirm you are on the latest TabShelf version.
2. Check existing open and closed issues.
3. Include reproducible steps.
