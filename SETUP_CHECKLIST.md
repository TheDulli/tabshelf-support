# Setup Checklist (GitHub-Only)

Use this checklist after creating your public `tabshelf-support` repository.

## 1) Upload files

Upload all files from this template, including hidden `.github/ISSUE_TEMPLATE/*` files.

## 2) Replace placeholders

- `YOUR_GITHUB_USERNAME`
- `ADD_YOUR_STORE_URL_HERE`
- Optional private security contact in `SECURITY.md`

## 3) GitHub repository settings

- Enable `Issues`
- Enable `Discussions` (optional)
- Keep source code repo private

## 4) Chrome Web Store listing fields

- Support URL: `https://github.com/YOUR_GITHUB_USERNAME/tabshelf-support`
- Homepage URL: `https://github.com/YOUR_GITHUB_USERNAME/tabshelf-support`
- Privacy policy URL: `https://github.com/YOUR_GITHUB_USERNAME/tabshelf-support/blob/main/PRIVACY.md`

## 5) Final verification

- Open `New issue` and confirm forms load without errors.
- Open `PRIVACY.md` URL in an incognito window to verify public access.
- Confirm no private source repository links are exposed.
